Overview

An integrated mobile robot system that connects low-level hardware, sensors, ROS 2 middleware, autonomous navigation, operating modes, and a user interface.

Engineering Focus

The main challenge was not a single algorithm, but coordinating multiple packages and interfaces into a predictable system. The project emphasizes clear package boundaries, state transitions, safety-oriented operation, and maintainable documentation.

Outcome

A reusable ROS 2 workspace structure for continued development of autonomous navigation, worker-following, and robot operation workflows.
